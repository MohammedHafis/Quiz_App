# 📝 Quiz API

A RESTful Quiz API built using Java and Spring Boot. This project allows users to create, manage, and attempt quizzes through API endpoints.

---

## 🚀 Features

- ➕ Create quiz questions
- 📋 View all questions
- 🎯 Submit answers
- 🌐 RESTful API architecture

---

## 🛠️ Technologies Used

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Maven
- H2 Database / MySQL
